/collect/start post

post

Starts a new session for processing data retreived from adapters. Calling /collect/data with token of the session will make worker request data from adapter with "query parameters" specifid at this initialization step

post /collect/start

Starts a new session for processing data retreived from adapters. Calling /collect/data with token of the session will make worker request data from adapter with "query parameters" specifid at this initialization step

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • HostName: required(string)

    Specify name of the host from wich you want to fetch measurement data. HostName has to be in adapter's hosts list

  • Dials: required(array of Dial)

    Specify the list of dials from which data will be measured. All dials have to be in adapter's dial list

    Items: Dial

    • Name: required(string)

      Name of the Dial

    • Unit: required(string)

      Unit of measurement

Example:

{
  "HostName": "host123",
  "Dials": [
    {
      "Name": "Processor_%",
      "Unit": "%"
    },
    {
      "Name": "Memory_%",
      "Unit": "%"
    }
  ]
}

HTTP status code 200

Returns session token

/collect/data post

post

Instructs worker to retrieve and process data from post /api/measure url of an adapter

post /collect/data

Instructs worker to retrieve and process data from post /api/measure url of an adapter

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • SessionID: required(string)

    Specify UUID of a session that is collected by requesting /collect/start

  • AdapterURL: required(string)

    Specify the URL of an adapter from which the data will be collected

Example:

{
  "SessionID": "cd920844-0fac-44e1-b39c-08234502c79b",
  "AdapterURL": "http://adapter-url/api/measure"
}

HTTP status code 200

Empty body

HTTP status code 400

Returns descriptive error string

Body

Media type: text/plain

Type: string

"Some error occured"
